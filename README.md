# DOFCAT: Dense Optical Flow CME Analysis Tool
A pipeline for CME detection and kinematic analysis using running-difference imaging and dense optical flow technique on coronagraph data (METIS onboard Solar Orbiter and ASPIICS onboard PROBA-3).

## Main features



* Reads FITS files and headers from a user-defined input directory.

* Applies instrument-specific masking.

* Applies temporal GTF filtering for ASPIICS to mitigate brightness flickering.

* Creates running-difference (RD) images using frame `i+2 - i`.

* Rotates ASPIICS RD images to solar-north-up using SunPy (in `DOFCAT_preprocessing_ASPIICS.py`). Not added here deliberately for METIS (`DOFCAT_preprocessing_METIS.py`), so users can see the difference. The correct approach would be to apply roll correction to METIS data as well.
 
* Uses header-based image dimensions and padding-aware cropping for defining the Optical Flow ROI.

* Computes dense optical flow for difference images using Farneback’s method.

* Converts pixel motion to physical velocity in km/s.

* Saves figures, headers, and compressed NumPy arrays in a clean output structure.

## Scripts



### 1. `DOFCAT_preprocessing_METIS.py`



This script processes METIS Total Brightness FITS files and creates pre-processed running-difference images.



#### What it does



1. Loads all FITS files from the input folder.

2. Reads the image data and headers.

3. Creates a valid field-of-view mask using instrument metadata.

4. Computes running differences using `I(i+2) - I(i)`.

5. Applies the mask to remove invalid pixels.

6. Computes plotting extent from WCS keywords.

7. Saves PNG images with consistent brightness scaling. It is an important step, detailed below.

8. Saves the aligned headers and compressed difference images.



#### Output



The script creates:



```text

save_dir/

└── difference_images/

    ├── difference_image_0000.png

    ├── difference_image_0001.png

    ├── ...

    ├── difference_headers.pkl

    └── diff_imgs.npz

```



#### Notes


* The figure size is derived from `NAXIS1` and `NAXIS2` in the FITS header.

* The output directory is user-defined.



---



### 2. `DOFCAT_preprocessing_ASPIICS.py`



This script processes PROBA-3/ASPIICS Wideband Level 2 FITS files and creates pre-processed & rotated running-difference images.



#### What it does
Similar to the METIS script, but with the GTF filter applied and roll angle corrected.


1. Loads all FITS files from the input folder.

2. Stacks the difference images into a time cube.

3. Applies a temporal Gaussian Tapered Fourier (GTF) filter for each pixel to reduce brightness flickering.

4. Computes running differences using `I(i+2) - I(i)`.

5. Applies an annular coronal mask and a solar-disk overlay.

6. Rotates each frame to solar north up using SunPy.

7. Saves PNG images with consistent brightness scaling. It is an important step, detailed below.



#### Output



The script creates:



```text

save_dir/

└── difference_images/

    ├── difference_image_0000.png

    ├── difference_image_0001.png

    ├── ...

    ├── difference_headers.pkl

    └── diff_imgs.npz

```



#### Notes



* The temporal GTF filter is preserved as part of the processing chain.

* Rotation and masking are applied after filtering.


---



### 3. `DOFCAT_OpticalFlowPipeline_METIS.py`



This script takes the running-difference images generated by the preprocessing stage and computes dense optical flow.



#### What it does


1. Loads the saved PNG difference images and converts them back into machine-readable arrays.

2. Applies bilateral filtering while preserving CME structure.

3. Defines a Region of Interest (ROI).

4. Computes dense optical flow using Farneback’s method.

5. Converts pixel motion into physical velocity using FITS metadata.

6. Filters unrealistic vectors using velocity thresholds.

7. Saves heatmaps with quiver arrows.

8. Saves vector overlays on the original frames.

9. Saves the `velocity magnitude`, `u` (horizontal velocity component), and `v` (vertical velocity component) for each pixel in a numpy file.



#### Output



The script creates:



```text

output_dir/

├── intensity/

│   ├── frame_0000.png

│   ├── frame_0001.png

│   └── ...

└── frames_with_vectors/

    ├── frame_0000.png

    ├── frame_0001.png

    └── ...

├── velocity_data/

│   ├── velocity_data.npz

```



#### Notes



* The optical-flow ROI is defined so that the padded border (which includes labels, titles, and ticks) is excluded.

## Running the scripts



### METIS preprocessing



```bash

python DOFCAT_preprocessing_METIS.py

```



Edit the input and output paths inside the script before running.



### ASPIICS preprocessing



```bash

python DOFCAT_preprocessing_ASPIICS.py

```



Edit the input and output paths inside the script before running.



### METIS optical flow



```bash

python DOFCAT_OpticalFlowPipeline_METIS.py

```



Edit the difference-image folder and output directory inside the script before running.
